### 1. 简介

多页签组件(基于应用平台)

### 2. 安装

```bash
npm install ac-multi-tabs -S
```

### 3. 使用
```javascript
import AcMultiTabs from 'ac-multi-tabs';
import 'ac-multi-tabs/dist/ac-multi-tabs.css';
```

```javascript
<AcNavbar />
```
